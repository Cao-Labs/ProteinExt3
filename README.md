# ProteinExt3

ProteinExt3 is an Fmax-optimized late-fusion framework for prospective protein function prediction. It integrates protein language model embeddings, BLAST-based homology transfer, and handcrafted physicochemical features under strict temporal and sequence-similarity-controlled evaluation.

## Main components

ProteinExt3 combines four prediction branches:

1. ESM-2 layer 20 embeddings
2. ESM-2 layer 33 embeddings
3. ProtT5 embeddings
4. BLAST-based homology transfer

The final prediction is generated by out-of-fold Fmax-optimized late fusion followed by GO DAG score propagation.

## Models

- ESM-2: `facebook/esm2_t33_650M_UR50D`
- ProtT5: `Rostlab/prot_t5_xl_half_uniref50-enc`

## Label spaces

Rare GO terms were filtered using aspect-specific thresholds:

| Aspect | Minimum count | Number of GO terms |
|---|---:|---:|
| Biological Process | 30 | 12,674 |
| Molecular Function | 15 | 5,077 |
| Cellular Component | 10 | 2,684 |

## Dataset summary

The training set contains 114,554 proteins and 628,834 raw experimental annotations. After GO ancestor propagation, the training set contains 4,301,975 propagated protein-term annotations.

The prospective test set contains 20,949 proteins and 30,977 annotations after temporal splitting and sequence-similarity filtering.

## Installation

```bash
conda create -n proteinext3 python=3.9 -y
conda activate proteinext3
pip install -r requirements.txt

python scripts/predict.py --in data/test.fasta --out predictions/ProteinExt3.tsv --method fusion

ProteinExt3/
├── proteinext3/
│   ├── training/
│   └── submethods/
├── scripts/

├── comparison/
├── docs/
├── requirements.txt
├── COMMANDS.md

This repository contains source code, preprocessing scripts, fusion weights, evaluation scripts, label-space files, and benchmark summary tables required to reproduce the ProteinExt3 workflow. Large model checkpoints, full embeddings, full prediction TSV files, and intermediate OOF artifacts are not included because of file-size limitations.
```
